2‚ÄÇOLS Intuition
Why invertibility matters

Visual road-map of Œ≤ÃÇ = (X‚Ä≤X)‚Åª¬πX‚Ä≤y


(p√óp) ‚Åª¬π X‚Ä≤y ‚Üí (X‚Ä≤X)‚Åª¬π √ó X‚Ä≤y = Œ≤ÃÇ

Invertibility üîë

  • Singular means some columns are linear combinations of others
  • If X‚Ä≤X is singular (rank < p) ‚Üí cannot invert ‚Üí no unique Œ≤ÃÇ.
  • Causes: perfect collinearity, dummy-variable trap, n < p, etc.
  • R‚Äôs lm() cleverly drops redundant columns, but still in left to right order.

3‚ÄÇErrors in lm, aov, lmer, lme

Simulated examples & fixes

lm() Error 1 ‚Äì Singular Fit

Message: ‚Äúcoefficients not defined because of singularities‚Äù

df <- data.frame(
  y  = rnorm(10),
  x1 = 1:10,
  x2 = 51:60,      # collinear with x1
  x3=31:40
)
lm(y ~ x1 + x2+x3, data = df)

Fix: drop redundant predictor

lm(y ~ x1, data = df)

lm() Lengths Differ

Message: ‚Äúvariable lengths differ‚Äù

x <- 1:10
y <- rnorm(9)
lm(y ~ x)          # unequal lengths

Error in model.frame.default(formula = y ~ x, drop.unused.levels = TRUE) : 
  variable lengths differ (found for 'x')

Fix: align vector lengths.

aov() Grouping Factor Has One Level

df <- subset(PlantGrowth, group == "ctrl")
aov(weight ~ group, data = df)   # fails

Error in `contrasts<-`(`*tmp*`, value = contr.funs[1 + isOF[nn]]) : 
  contrasts can be applied only to factors with 2 or more levels

Fix: ensure ‚â• 2 levels.

lmer() Error 1 ‚Äì One-Level Random Effect

Message: ‚Äúnumber of levels of a grouping factor is 1‚Äù

library(lme4)
df <- data.frame(
  y = rnorm(10),
  x = rnorm(10),
  group = rep("A", 10)
)
lmer(y ~ x + (1 | group), data = df)

Error: grouping factors must have > 1 sampled level

Fix: add more group levels.

4‚ÄÇRank Deficiency & Dummy-Variable Trap

Perfect collinearity demo

set.seed(1)
d <- data.frame(
  y = rnorm(40),
  x1 = rnorm(40)
)
d$x2 <- d$x1        # duplicate
lm(y ~ x1 + x2, d)  # x2 dropped (NA coef)

Dummy-variable trap

iris$Species <- factor(iris$Species)
X.bad <- cbind(1, model.matrix(~ Species + 0, iris))  # intercept + all dummies
qr(X.bad)$rank  # 3 < 4 ‚áí singular

Fix by dropping one level (~ Species) or omitting intercept (~ Species + 0).

5‚ÄÇMulticollinearity & Numerical Instability

Longley example

data(longley)
m <- lm(Employed ~ ., longley)
kappa(m)        # ‚â´ 10^4 : ill-conditioned

High condition # ‚Üí unstable estimates.

Floating-point quirks

(1e16 + 1) - 1e16       # 0
sqrt(2)^2 - 2           # 4.4e-16
  • Rescale predictors; center & scale.
  • Use ridge / lasso for severe collinearity.
